## Overview

CDPaint is built for creators who want a simple, familiar pixel editing experience while still keeping the tools needed for retro-focused production. Its interface is modeled after the MS Paint style, which makes it easy to pick up for fast edits, sprite work, and indexed artwork.

The project fits especially well into workflows for classic handheld assets, ROM hacking, and other low-color production pipelines. Because it runs in both web and desktop environments, CDPaint offers a practical way to draw, refine, and export pixel art wherever your project lives.

## How to use CDPaint

Begin by creating or importing pixel art, then choose the workflow that best matches the task at hand:

- Use the drawing tools for direct pixel placement and MS Paint-style editing
- Adjust indexed colors in the palette editor when working with limited-color assets
- Split sprite sheets into smaller pieces before exporting or organizing assets
- Use selection tools to move, edit, or isolate sections of your artwork
- Switch on tiled drawing mode when building repeating patterns or grid-based assets
- Assign custom hotkeys to speed up repetitive editing tasks
- Export assets using the retro and GBA-oriented output tools when preparing game content

For the cleanest results, keep your palette setup and export target aligned with the format you need before wrapping up a project.
